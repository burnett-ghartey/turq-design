## ğŸ“ Customization

### Update Content

All content is currently placeholder text generated by AI. Update the following in `src/pages/index.js`:

- Service descriptions
- Project details
- Stats numbers
- About section text
- Contact information in Footer.js

### Replace Images

Add your own project images to the `/public` folder and update image paths throughout the components.

### Modify Colors

Update brand colors in `src/styles/globals.css`:

```css
:root {
  --turq-yellow: #f9d412;
  --turq-cream: #f1f2de;
  --turq-teal: #20807e;
}
```
